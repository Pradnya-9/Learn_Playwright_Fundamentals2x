# Learn Playwright Fundamentals 2x

This repository contains a Playwright test project created with Playwright Test.

## Project contents

- `package.json` - project metadata and dev dependencies
- `playwright.config.ts` - Playwright test configuration
- `tests/example.spec.ts` - example Playwright tests
- `.gitignore` - excludes `node_modules/` and Playwright artifacts

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the tests:
   ```bash
   npm test
   ```

## Notes

- The `node_modules/` folder is excluded from source control via `.gitignore`.
- This repo is intended for Playwright fundamentals and sample browser tests.
